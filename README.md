# Zlither
## A Desktop Client for Slither.io

Zlither is a client for [Slither.io](https://slither.io) which can run outside the browser. It is written in C and uses Vulkan for rendering all the graphics, resulting in faster rendering. This is a fork of Vlither client that has been made to be easier to use and have a few more features!

### Running Instructions
Just download the executable file from [here](https://zlither.pages.dev/) and run it!

## Features

- Upscaled/sharper textures
- Supports all 66 skins and 32 accessories
- Custom skin
- Server selection
- Zoom
- Kill count
- Bot ([Saya's implemenation](https://github.com/saya-0x0efe/Slither.io-bot))
